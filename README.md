# الباش مهندس ناصر - فني صيانة موبايل احترافي

مرحباً بكم في مشروعي الأول على GitHub. أنا متخصص في تقديم خدمات الصيانة الشاملة للهواتف المحمولة.

## 🛠️ الخدمات التي أقدمها:
* *صيانة هاردوير:* تشخيص وإصلاح أعطال البوردة والدوائر الإلكترونية بدقة.
* *صيانة سوفت وير:* تفليش الرومات، حل مشاكل النظام، وتخطي الحسابات.
* *تركيب باغات:* تغيير الباغات والشاشات المكسورة بأحدث الأجهزة والتقنيات.
